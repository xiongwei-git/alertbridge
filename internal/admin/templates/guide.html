CANONICAL EVENT API

统一事件接口

这是 AlertBridge 唯一的正式入站契约。调用方负责把自身事件映射成下面的统一结构，再对原始 JSON 正文签名。

POST {{.GuideBaseURL}}/api/v1/events
  • 正文必须明确包含 routing_key
  • 服务可靠写入后返回 202 Accepted
  • 原始 JSON 正文上限为 32 KiB

AUTHENTICATION

签名规则

所有调用方都使用这一套认证方式。

1

计算正文摘要

hex(SHA256(raw_body))
→
2

拼接待签名文本

timestamp\nnonce\nbody_digest
→
3

使用客户端密钥

hex(HMAC-SHA256(secret, text))
请求头用途
X-Notify-Client创建客户端时填写的 Client ID
X-Notify-Timestamp当前 Unix 秒级时间戳，必须在服务器允许时间窗内
X-Notify-Nonce每次请求使用新的 8–128 位随机字符串，防止重放
X-Notify-Signature按上方规则计算的 64 位十六进制 HMAC-SHA256
不要先签名再修改正文。空格、换行或字段顺序发生变化都会导致签名失效。密钥应放在环境变量或 Secret 文件中，不要写进 Git、日志或前端代码。

READY-TO-RUN EXAMPLES

调用示例

当前服务地址、客户端和可用路由已自动代入；只需粘贴刚才保存的密钥。

Shell：curl + OpenSSL
BASE_URL='{{.GuideBaseURL}}'
CLIENT_ID='{{.GuideClientID}}'
CLIENT_SECRET='粘贴创建客户端时保存的密钥'
ROUTING_KEY='{{.GuideRoute}}'

timestamp=$(date +%s)
nonce=$(openssl rand -hex 12)
occurred_at=$(date -u +%Y-%m-%dT%H:%M:%SZ)
body=$(printf '{"event_id":"manual-%s-%s","source":"manual-test","routing_key":"%s","status":"test","severity":"{{.GuideSeverity}}","title":"AlertBridge 接入测试","message":"外部客户端签名调用成功。","occurred_at":"%s"}' "$timestamp" "$nonce" "$ROUTING_KEY" "$occurred_at")
body_digest=$(printf '%s' "$body" | openssl dgst -sha256 -hex | awk '{print $NF}')
signature=$(printf '%s\n%s\n%s' "$timestamp" "$nonce" "$body_digest" | openssl dgst -sha256 -hmac "$CLIENT_SECRET" -hex | awk '{print $NF}')

curl --fail-with-body --silent --show-error \
  -H 'Content-Type: application/json' \
  -H "X-Notify-Client: $CLIENT_ID" \
  -H "X-Notify-Timestamp: $timestamp" \
  -H "X-Notify-Nonce: $nonce" \
  -H "X-Notify-Signature: $signature" \
  --data-binary "$body" \
  "$BASE_URL/api/v1/events"
Python：仅使用标准库
import hashlib
import hmac
import json
import secrets
import time
import urllib.request
from datetime import datetime, timezone

BASE_URL = "{{.GuideBaseURL}}"
CLIENT_ID = "{{.GuideClientID}}"
CLIENT_SECRET = "粘贴创建客户端时保存的密钥"

timestamp = str(int(time.time()))
nonce = secrets.token_hex(12)
payload = {
    "event_id": f"python-{timestamp}-{nonce}",
    "source": "python-example",
    "routing_key": "{{.GuideRoute}}",
    "status": "test",
    "severity": "{{.GuideSeverity}}",
    "title": "AlertBridge Python 接入测试",
    "message": "Python 标准库签名调用成功。",
    "occurred_at": datetime.now(timezone.utc).isoformat(),
}
body = json.dumps(payload, ensure_ascii=False, separators=(",", ":")).encode()
body_digest = hashlib.sha256(body).hexdigest()
signing_text = f"{timestamp}\n{nonce}\n{body_digest}".encode()
signature = hmac.new(CLIENT_SECRET.encode(), signing_text, hashlib.sha256).hexdigest()

request = urllib.request.Request(
    f"{BASE_URL}/api/v1/events",
    data=body,
    method="POST",
    headers={
        "Content-Type": "application/json",
        "X-Notify-Client": CLIENT_ID,
        "X-Notify-Timestamp": timestamp,
        "X-Notify-Nonce": nonce,
        "X-Notify-Signature": signature,
    },
)
with urllib.request.urlopen(request) as response:
    print(response.read().decode())
